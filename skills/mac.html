Macintosh Experience

I've been working with Macintoshes since they were first released in 1984. I've developed a wide range of applications for Macs, primarily education related products, but also production and art utilities. Some of the products I've developed for Macs include:

Starry Night Dome

Simulation Curriculum hired me to do some additional upgrades to the Starry Night C++ code, including preparing their Dome product for a new release.   [details...]

Scholastic Data Migration Utility

I created this utility for Scholastic, Inc. in C++ for both Windows and Mac OSX. It enabled Scholastic's customers to migrate their Read 180 data.   [details...]

Scholastic Read 180 Edit Tools

I upgraded a set of Macintosh-based production tools written in C++ for Scholastic, Inc.   [details...]

Starry Night Localization

Imaginova hired me to prepare their successful astronomy project, Starry Night, for localization. I updated the C++ code, and developed a series of perl scripts for the process.   [details...]

Interactive Phonics System

Scholastic, Inc. hired me to create a 12-CD set of interactive stories and activities in Director, along with production tools written in C++ and Quicktime, Director, Delphi, and perl.   [details...]

HyperTV Macintosh Plugin

ACTV hired me to port their HyperTV internet browser plugin written in C++ that I ported from Windows to the Macintosh.

TenCORE Net Macintosh Development

I developed some of the Macintosh-specific modules of TenCORE Net, an ambitious multi-platform web language project created in C, for Catharon Multimedia Productions. You can check out the language, now called VDelta at vdelta.com

Speech Recognition Prototype

A windows-based prototype for a speech recognition product which I developed for Scholastic, Inc., using C++, IBM's ViaVoice engine, and Lernout and Hauspie's voice recognition engine.

eSchool ActiveX Control

ACTV hired me to create this Macintosh ActiveX control for Internet Explorer in C.

Get Set For Kindergarten

Children's Television Workshop asked me to port their windows product, written in C++, to the Macintosh.

A Field Trip to the Sky

A consumer astronomy product, programmed in Director, with plugins programmed in C for Windows and Macintosh, which I developed for Terra Associates on behalf of Sunburst.

SyncLips

Children's Television Workshop hired me to design and develop this audio editing utility in C for the Macintosh.

CUNY Math Survey

CUNY hired me to port an interactive math survey they had developed for DOS to the Macintosh and AppleII.
