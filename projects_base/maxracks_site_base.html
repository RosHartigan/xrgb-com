M@xracks Website

In 1999Maxracks hired me to port www.maxracks.com, an e-card site, from a Macintosh to a unix server.  It was a complex website, using a 4th Dimension database server to log users, store an addressbook for each user, and generate a pict file on the fly that combined an image the user had chosen with a message from the user, a stamp, and an email address.  I recreated the site for unix using perl as the CGI language, and mSQL as the database server.  I also reprogrammed their redesigned home page in HTML and javascript.  I finished the port in 1999.  In 2002, they hired me again to upgrade their site to new versions of the perl GD module, and to switch from mSQLto MySQL as the database server.
