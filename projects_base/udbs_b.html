UDBSPlus Utility Billing System

In 2011, Open Platform Group hired me to assist in the port and development of a utility billing system, UDBSPlus. UDBSPlus is a port of a product written in basic, with some additional improvements and redesign. It is a Windows client written in C++ and wxWidgets, using MySQL to handle the data.
