Math-O-Matic

Mullaghdun asked me to rewrite their successful Math-O-Matic product in order to make it more expandable. I redesigned and rewrote the Director code from the ground up, including a set of libraries I designed that strengthened Lingo's object-oriented abilities. We also developed the code for a new series of math modules dealing with fractions and integers.
