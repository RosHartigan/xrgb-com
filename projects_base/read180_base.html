Read 180 Demo

In 1999, Scholastic, Inc. hired me to create the Read 180 demo.  I programmed the demo in Director. 

This was a fully interactive demo that allowed the user to test each part of the actual program.  I used marked audio and a database of word decodings to generate the phonics portion of the demo.

I also developed a number of production utilities in Director to assist the Scholastic team in creating the Read 180 assets.

I created this Read 180 demo for Scholastic, Inc. using Director.
