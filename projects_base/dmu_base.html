Scholastic Data Migration Utility

In late 2004, Grassroots Technologies asked me to create a utility for Scholastic, Inc. They needed a tool to enable their customers to migrate Read 180 data into a new format. I developed it in C++ for both Windows and Mac OSX.

I created this utility for Scholastic, Inc. in C++ for both Windows and Mac OSX. It enabled Scholastic's customers to migrate their Read 180 data.
