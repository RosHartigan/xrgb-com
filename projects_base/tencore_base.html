TenCORE Net Macintosh Development

In 1999, as part of a large, distributed programming team, I developed some of the Macintosh-specific modules of an ambitious multi-platform web language project created in C, called TenCORE Net.  This project was created and managed by Catharon Multimedia Productions.  You can check out the language, now called VDelta at vdelta.com.
