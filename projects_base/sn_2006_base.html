Starry Night Localization

In 2005 Imaginova approached me about working on their successful astronomy product, Starry Night, and its associated telescope plug-ins. I upgraded the C++ code to be unicode friendly, as well as updating the user interface to allow for the insertion of variable length text. To assist in the process, I prepared a series of perl scripts to extract, log, translate, and reinsert text. Imaginova released the German version of Starry Night in 2006.
