Scholastic Read 180 Edit Tools

In late 2003, Grassroots Technologies came to me on behalf of Scholastic, Inc. In preparation for a new release of Read 180, they asked me to overhaul and make useable a set of production tools written for the Macintosh in C++. I was able to give them a working set of tools by early 2004.
