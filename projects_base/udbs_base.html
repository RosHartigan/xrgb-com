UDBS-Plus Utility Billing System

In 2011, Open Platform Group hired me to assist in the port and development of a utility billing system. UDBS-Plus is a port of a product written in basic, with some additional improvements and redesign. is a Windows client written in C++ and wxWidgets, using MySQL to handle the data.

Open Platform Group hired me to assist in the port of a billing system to a C++ Windows client using MySQL.
